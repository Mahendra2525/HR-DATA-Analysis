### HR Analytics Dashboard
An interactive Power BI/Excel dashboard designed to track and analyze employee attrition.
The project provides actionable insights into workforce distribution, turnover trends,and 
demographic patterns to assist HR departments in improving employee retention.

#### Key Features (Bullet points)
KPI Tracking: Monitors Total Employees, Attrition Count, and Attrition Rate (14.80%).
Demographic Analysis: Visualizes attrition across gender, age groups, and marital status.
Departmental Breakdown: Identifies turnover hotspots across HR, R&D, and Sales departments.
Educational Insights: Correlates educational background with attrition rates.
Interactive Filtering: Includes slicers for Education Field and Department to allow dynamic data exploration.
